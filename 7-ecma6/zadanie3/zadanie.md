11. Przygotuj dwa obiekty:
    1.  Obiekt 1 ma pola: { id: number, text: string }
    2.  Obiekt 2 ma pola: { id: number, description: string, details: { extraText: string } }
12. Skopiuj obydwa obiekty za pomocą operatora spread.
